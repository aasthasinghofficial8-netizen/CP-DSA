# [3.Josephus problem](https://www.geeksforgeeks.org/problems/josephus-problem/1)

## Description
<p>
You are playing a game with n people standing in a circle, numbered from 1 to n. Starting from person 1, every kth person is eliminated in a circular fashion. The process continues until only one person remains.
Given integers n and k, return the position (1-based index) of the person who will survive.
Constraints:
1 ≤ n, k ≤ 500
</p>

## Solution (CPP)
<p>
<ol>
<li>Using erase method</li>


<!-- tabs:start -->
```

class Solution {
public:
    int josephus(int n, int k) {
        vector<int> ind;
        for(int i=1;i<=n;i++){
            ind.push_back(i);
        }
        int len=ind.size();
        int currentpos=0;
        while(len>1){
            int target=(currentpos+k-1)%ind.size();
            ind.erase(ind.begin()+target);
            currentpos=target%ind.size();
            len-=1;
        }
        return ind[0];
    }
};


```
<!-- tabs:end -->
